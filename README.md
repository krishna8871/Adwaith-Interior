# 🏡 Adwaith Interiors - PDF Quotation Generator

A custom, progressive web application (PWA) built with Google Apps Script to streamline the creation, management, and storage of interior design quotations.

## ✨ Features

* **Dynamic PDF Generation:** Instantly converts form inputs into a professionally formatted, downloadable PDF quotation with embedded company branding.
* **Automated Calculations:** Automatically calculates Square Footage (SqFt), Gross Amounts, Subtotals, and Grand Totals for both Wood Work and Fittings & Fixtures.
* **Smart Cloud Storage:** Automatically saves generated PDFs directly to Google Drive, intelligently organizing them into dynamically created `Year / Month` folders.
* **Database Logging:** Logs every generated quotation into a connected Google Sheet.
* **Internal Notifications:** Automatically emails the internal team whenever a new quotation is generated.
* **Quotation History & Editing:** Features a built-in dashboard to view past quotations, share links via WhatsApp, and load previous data back into the form for quick editing.
